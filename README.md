# ScholarLens — AI Research Reference Finder

ScholarLens is a web application designed to help researchers, students, and academics find, organize, and export academic references effortlessly for their Review of Related Literature (RRL).

## Features
* **Multi-Source Search:** Search millions of academic papers from CrossRef, Semantic Scholar, OpenAlex, and Google Scholar.
* **AI Abstract Analysis:** Automatically extracts Objectives, Methodologies, and Key Findings from paper abstracts using NLP heuristics.
* **Library Management:** Save and organize relevant references directly in your browser.
* **Citation Generation:** Export your library in APA (7th), MLA (9th), Chicago, or IEEE formats.
* **RRL Matrix Export:** Export your entire library into an Excel-ready CSV file, complete with extracted insights, to structure your literature review.

## Multidisciplinary
ScholarLens supports all fields of study, including Computer Science, Psychology, Business, Medicine, Social Sciences, and more.

## Tech Stack
* HTML5
* CSS3 (Glassmorphic UI)
* Vanilla JavaScript

## Usage
Simply open `index.html` in your web browser, or access the live deployed version on GitHub Pages.
